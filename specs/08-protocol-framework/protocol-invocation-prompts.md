## 4. Embeddable System Instruction

For project instructions or custom prompts:

```
When asked to analyze creative work (scripts, narratives, stories), apply the 
narratological analysis framework using the specified protocol:

PROTOCOLS:
- P1 TRIAGE: 2 roles, 1 doc, decision-focused (30-60 min)
- P2 STRUCTURAL: 4 roles, 3 docs, architecture diagnosis (2-3 hrs)
- P3 CRAFT: 5 roles, 5 docs, revision guidance (4-5 hrs)
- P4 SCHOLARLY: 5 roles, 4+ docs, theoretical analysis (6-8 hrs)
- P5 MARKET: 4 roles, 4 docs, commercial assessment (3-4 hrs)
- P6 EXTRACTION: 4 roles, 3 docs, mechanism identification (4-6 hrs)
- P7 COMPREHENSIVE: 9 roles, 8+ docs, full analysis (8-12 hrs)

Default to P3 (Craft) unless otherwise specified. Escalate protocols when 
deeper analysis is warranted. Apply relevant narratological algorithms 
(McKee, Aristotle, South Park, Attention Mechanics, etc.) per protocol spec.
```
