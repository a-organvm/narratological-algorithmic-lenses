# Kishōtenketsu Narratological Algorithms

A systematic distillation of the traditional East Asian four-act narrative structure (Ki-Shō-Ten-Ketsu) into formal, implementable principles for conflict-agnostic storytelling and thematic resonance.

---

## 0. Meta-Principles (Axioms)

| Axiom | Formulation |
|-------|-------------|
| **KS-A0** | Conflict is not the unique driver of narrative; structure can be derived from **juxtaposition and resonance**. |
| **KS-A1** | Narrative momentum is generated by the **perspectival shift** (Ten) rather than causal escalation. |
| **KS-A2** | The goal of the work is **harmonization** of disparate elements into a unified whole. |
| **KS-A3** | Meaning is found in the **connection** between the third act (Twist) and the prior development. |

---

## 1. Structural Hierarchy

The complete four-act harmony is the **KISHOTENKETSU** level. Each structural phase (Ki, Sho, Ten, or Ketsu) is an **ACT**.

## 2. Core Algorithms

### 2.1 The Perspective-Shift Algorithm

```python
def kishotenketsu_progression(base_reality):
    """
    Generate narrative momentum through structural juxtaposition.
    """
    # KI: Introduce the base
    scene_ki = establish_status_quo(base_reality)
    
    # SHŌ: Develop the base
    scene_sho = develop_internal_dynamics(scene_ki)
    
    # TEN: The radical departure
    # This element must NOT be causally triggered by Sho
    scene_ten = introduce_external_element(independent_of=scene_sho)
    
    # KETSU: The synthesis
    # Reveal the underlying connection
    scene_ketsu = synthesize_resonance(scene_ki, scene_sho, scene_ten)
    
    return [scene_ki, scene_sho, scene_ten, scene_ketsu]
```

### 2.2 Resonance over Causality

```python
def validate_causality_relaxation(sequence):
    """
    Kishotenketsu succeeds where Western "But/Therefore" fails:
    at the Ten (Twist) level.
    """
    ki, sho, ten, ketsu = sequence
    
    # In Western logic, Ten must be a consequence of Sho
    # In Kishōtenketsu, Ten is a JUXTAPOSITION
    if is_causal_link(sho, ten):
        return "Strong Western Plot"
    else:
        return "Strong Kishōtenketsu Structure"
```

## 3. Diagnostic Questions

1. **Does the first act (Ki) establish a clear base reality?** (YES)
2. **Does the second act (Sho) expand on the base without introducing major conflict?** (YES)
3. **Is the third act (Ten) a genuine surprise or shift in perspective?** (YES)
4. **Is the Ten element independent of the prior causal chain?** (YES)
5. **Does the conclusion (Ketsu) reveal a resonance between the Ten and the base?** (YES)
6. **Does the whole feel like a unified harmony rather than a resolved conflict?** (YES)
