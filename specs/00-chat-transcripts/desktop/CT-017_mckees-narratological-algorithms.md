**Structural summary:**

1. **Meta-principles** — Principles over rules; archetypes over stereotypes; form over formula

2. **Structural hierarchy** — Beat → Scene → Sequence → Act → Story; each level containers the one below

3. **The Story Triangle** — Maps the design space: Archplot (classical), Miniplot (minimalism), Antiplot (anti-structure)

4. **The Gap algorithm** — The core engine: character takes minimal action expecting cooperation → reality contradicts expectation → gap opens → must escalate risk. This repeats until climax.

5. **Value charge system** — Every unit operates on binary value pairs (life/death, love/hate, etc.); scenes must turn values or be cut

6. **Three-level conflict architecture** — Inner (mind/body/emotion) → Personal (intimate relationships) → Extra-personal (institutions/environment)

7. **Inciting Incident protocol** — Must radically upset balance; protagonist must be aware; creates the Spine (conscious + unconscious desire)

8. **Act design formulas** — Three-act minimum; proportions (25%/50%/15%/10%); subplot solutions for long second acts

9. **Scene validity test** — `IF opening_value == closing_value: DELETE`

10. **Composition** — Unity (causal lock between Inciting and Climax) + Variety + Pacing (alternating tension/release)

11. **Exposition** — Never include what audience can assume; dramatize, don't declare

12. **Character-structure binding** — True character revealed under pressure; empathy required, sympathy optional

13. **Controlling Idea** — Single sentence: `VALUE + CAUSE`

14. **Minimum conservative action** — Characters always take smallest action they believe sufficient

15. **Risk calculus** — Desire value proportional to risk; if failure = "return to normal," story not worth telling

16. **Consistency rules** — Internal rules of fictional world cannot be violated once established
